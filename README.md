## Beginner Projects

These projects focus on fundamental machine learning concepts including
data preprocessing, exploratory data analysis, and basic predictive
modeling.

### Heart Disease Prediction

A machine learning project analyzing healthcare data to identify factors
associated with heart disease.

Key components: - Data cleaning and preprocessing - Exploratory data
analysis - Feature engineering - Classification models

Algorithms used: - Logistic Regression - Decision Trees - Random Forest

------------------------------------------------------------------------

### Movie Rating Analysis

A data analysis and predictive modeling project examining patterns in
movie rating datasets.

Key components: - Data visualization and exploratory analysis - Feature
correlation analysis - Predictive modeling experiments

Techniques used: - Linear Regression - Feature encoding - Exploratory
data analysis

------------------------------------------------------------------------

### Premier League Performance Analysis

A sports analytics project analyzing performance data from Premier
League teams to identify patterns and performance trends.

Key components: - Statistical analysis of match data - Team performance
comparison - Visualization of performance trends

Techniques used: - Data aggregation - Feature analysis - Predictive
modeling experiments

## Intermediate Projects

These projects involve more complex machine learning systems including
full-stack applications, data pipelines, and LLM-based systems.

### ML Stable Diffusion Dashboard

A full-stack analytics dashboard designed to analyze GitHub repository
activity related to Stable Diffusion projects.

System components: - Python data collection pipeline - Backend API for
analytics - Frontend dashboard for visualization

Architecture:

    Frontend (React)
    ↓
    Backend API (Node.js / Express)
    ↓
    Data Collector (Python)
    ↓
    GitHub API

Technologies used: - Python - Node.js / Express - React - GitHub API -
SQL database

------------------------------------------------------------------------

### RAG Agent using LangGraph

A retrieval-augmented generation (RAG) system built using LangGraph. The
project demonstrates how LLM-based agents retrieve relevant information
from a vector database before generating responses.

System components: - Retrieval pipeline - LangGraph agent architecture -
Vector database integration - Modular agent node design

Architecture:

    User Query
    ↓
    Retriever (Vector Database)
    ↓
    LangGraph Agent
    ↓
    LLM Response

Technologies used: - Python - LangGraph - Vector databases - Retrieval
pipelines
